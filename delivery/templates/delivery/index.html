Articles

Articles will be submitted directly to Illiad. 360Link will run in the background and display holdings links if it can. Otherwise a request button will be provided.

  • Article held electronically
  • Article not held
  • Article held in print.
  • Journal level link. Not direct to full text.
  • Journal level link referred by OCLC. Not held.
  • Journal level link referred by Vufind. Held online.
Articles lacking metadata
  • Article without starting page number.
  • Article missing metadata.
Pubmed requests

These requests will pull metadata from the Pubmed API to get citation information.

  • Pubmed request - item held.
  • Pubmed request - item not held.
  • Pubmed request - no direct link.
  • Pubmed Central artilce. Full text provided directly from Pubmed API.
